# Change：教材学习改为左侧目录 + 按册筛选

**日期**：2026-07-03

## 需求

教材学习页面在左侧提供课程目录，并支持按教程（新概念 1–4 册）切换：选中哪一册，目录只展示该册的课程。

## 改动内容

### 前端

- `public/index.html`
  - 教材学习区域重构为左右布局（`.learn-layout`）：左侧边栏（册数切换 chips + 课程目录列表），右侧为课程详情区。
  - 移除原独立的课程详情面板 `lessonDetailPanel` 和"返回目录"按钮（目录常驻左侧，不再需要返回）；"练本课语法"按钮保留在详情顶部。
  - 右侧未选课时显示占位提示。

- `public/app.js`
  - 新增 `state.learnBook`（教材学习当前册数，默认第 1 册）。
  - 新增 `renderLearnBookChips()`：渲染左侧"新概念 1–4"切换按钮，切换时重新加载目录并清空右侧详情。
  - `loadLessons(book)`：由写死 `book=1` 改为按参数加载；渲染为侧栏目录条目（课号 + 英文标题 + 中文标题）；该册无数据时显示"暂未收录"。
  - `openLesson()`：详情渲染到右侧内容区，同时高亮左侧目录当前课。
  - `practiceCurrentLesson()`：取题接口由写死 `book=1` 改为跟随当前课所属册数（顺带修复）。

- `public/styles.css`
  - 新增 `.learn-layout / .learn-sidebar / .learn-toc / .toc-item` 等样式；侧栏 sticky 定位、目录独立滚动、当前课高亮。
  - 移动端（≤760px）侧栏折叠为顶部全宽，目录限高 42vh。
  - 删除已无处使用的 `.lesson-list / .lesson-card` 旧卡片网格样式。

### 后端

无改动。`/api/lessons?book=N` 原本已支持按册筛选。

## 验证

Puppeteer 无头浏览器端到端验证通过：

1. 进入教材学习，左侧显示新概念 1–4 切换项，第 1 册目录 72 条，右侧显示占位提示。
2. 点击目录项，右侧展示课程详情，对应目录项高亮。
3. 切到新概念 2（暂无数据），目录显示"该册课程内容暂未收录"，右侧详情清空回占位。
4. 切回新概念 1，目录恢复 72 条。

## 备注

- 目前仅第 1 册收录了课程内容（72 课）；第 2–4 册数据补充到 `data/lessons/` 分片后，目录会自动生效，无需再改前端。
