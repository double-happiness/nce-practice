# 第一、二册收尾：功能对齐 + 第二册精读短文

日期：2026-07-03

## 背景

第二册 96 课教材与题库虽已齐全，但产品层仍有明显短板：听写只支持第一册、背单词/词汇量测试册数写死、第二册缺少精读短文（教材学习「原文」标签空白）、页脚与 README 仍写「仅第一册完整」。

本轮在**不启动第三册**的前提下，把第一、二册的学习闭环对齐到同一水准。

## 改动

### 1. 澄清第一册覆盖口径

新概念第一册 144 课中，**奇数课为课文、偶数课为练习**。系统按课文课收录，**72 课即全书内容完成**（非 72/144 未完成）。

### 2. 第二册精读短文（`data/articles/b2-*.json`）

新增 8 个分片，共 **96 篇**原创英中对照短文（每课 6–10 句），与课文情景和语法点对齐：

- `b2-001-012` … `b2-085-096`

`lib/data.js` 自动合并到课程 `article` 字段，教材学习页「原文」标签与听写句源可用。

### 3. 听写支持第二册（`feat-dictation.js`）

- 新增册数选择（第 1 / 2 册）
- 教材页「听写本课」深链对第二册生效
- 听写句集**优先使用精读短文**，其次单词例句与语法例句
- 断点记忆键改为 `册-课`，避免两册课号冲突

### 4. 背单词 / 听力词汇量 / 阅读词汇量支持第二册

- `feat-words.js`：顶栏册数切换（第 1 / 2 册）
- `feat-listenvocab.js`、`feat-readvocab.js`：入口页可选册数
- 首页「待巩固单词」合并统计第 1、2 册

### 5. 文案更新

- `README.md`：内容覆盖表与第一册口径说明
- `index.html` 页脚：第一、二册已完整收录

- B1 精读：72 / 72 课 · B2 精读：96 / 96 课
