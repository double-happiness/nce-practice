# 词典增强：固定搭配 · 课内句型 · 教材例句

日期：2026-07-03

## 功能

查词典时，每个词条可展开「搭配 · 句型 · 例句」详情：

- **固定搭配 / 短语**：从该词教材例句与语法示例中自动提取含目标词的短语
- **课内常用句型**：该词出现过的课文中的 `grammar[]` 语法点（标题、讲解、示例）
- **教材例句**：全册扫描该词所有 `words[].eg`，去重汇总并标注出处课次

高频易混短语可在 `data/word-enrich.json` 手工补充（如 `look forward to`、`have to`、`be used to`）。

## API

- `GET /api/words/detail?word=xxx` — 返回 `{ word, phon, pos, cn, collocations, patterns, examples, sources }`

## 涉及文件

- `lib/word-detail.js` — 聚合逻辑
- `routes/words.js` — 详情接口
- `public/js/feat-dictionary.js` — 展开/懒加载 UI
- `data/word-enrich.json` — 可选手工补充
