# 词典增强：固定搭配 · 课内句型 · 教材例句

日期：2026-07-03

## 功能

查词典时，每个词条可展开「搭配 · 句型 · 例句」详情：

- **固定搭配 / 短语**：从该词教材例句与语法示例中自动提取含目标词的短语
- **课内常用句型**：该词出现过的课文中的 `grammar[]` 语法点（标题、讲解、示例）
- **教材例句**：全册扫描该词所有 `words[].eg`，去重汇总并标注出处课次

高频易混短语可在 `data/word-enrich.json` 手工补充（如 `look forward to`、`have to`、`be used to`）。

## API

- `GET /api/words/detail?word=xxx` — 返回 `{ word, phon, pos, cn, collocations, patterns, examples, sources }`

## 涉及文件

- `lib/word-detail.js` — 聚合逻辑
- `routes/words.js` — 详情接口
- `public/js/feat-dictionary.js` — 展开/懒加载 UI
- `data/word-enrich.json` — 可选手工补充

## 补充数据（2026-07-03 扫描生成）

- **batch1**：62 条 — 全部多词短语 + 易混语法锚点
- **batch2**：扩展至 **953 条** — 覆盖 B1+B2 全部 908 个教材词条 + 45 条语法短语
- **batch3**：+35 条 B2 动词搭配与情态动词句型（accuse/adjust/escape/worth/prefer/need/can…）及 `not only…but also`、`every time`、`may have done` 等
- 手工表：`word-enrich-manual.json` + `batch2` + `batch3`（batch2 与 batch3 重复键时 batch2 优先）
- 生成：`npm run build:word-enrich`
- 校验：`npm run validate` 会检查 `word-enrich.json` 结构

## 体验优化（同日）

- 精确搜到唯一词条时**自动展开**搭配详情；错词复习模式亦自动展开
- **深链查词**（课文 📕、听写点词、首页快捷查词等）自动展开精确匹配词条（多结果时也能定位到目标词）
- 教材例句出处可点击跳转对应课文
- 全局词库词条展开时显示基础释义（无教材搭配时有明确提示）
- `word-detail.js`：手工 enrich 搭配/句型优先于课内自动聚合
