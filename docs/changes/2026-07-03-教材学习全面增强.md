# Change：教材学习全面增强——进度、搜索、正确率、掌握标记、生词收藏、听写深链

**日期**：2026-07-03（书页式目录的第三轮迭代，前两轮见同目录另两篇文档）

## 背景

用户授权大胆完善。围绕"一本真的教材该怎么用"补齐学习闭环：学（教材）→ 练（题库）→ 记（生词本）→ 听（听写）→ 复盘（正确率回流到目录）。

## 新增功能

### 目录页（左页）

1. **学习进度条**：页眉下方显示"已学 n / N 课 · 已掌握 m 课 ★"，金红渐变条随已学比例增长。
2. **目录搜索**：纸面样式搜索框，按课号 / 英文标题 / 中文标题实时过滤；切换册数自动清空。
3. **每课练习正确率角标**：目录条目中文标题右侧显示该课题目正确率（绿 ≥80 / 黄 60–79 / 红 <60），练过才显示。数据来自 `/api/stats/lesson`。
4. **三态标记**：✓ 已学（自动，绿色）→ ★ 已掌握（手动，金色，覆盖 ✓）。

### 内容页（右页）

5. **本课学习情况条**：标题下方显示"本课已练 X 次 · 正确率 Y% · 重点单词 N 个 · 语法点 M 个"。
6. **☆ 标为已掌握**：顶部按钮一键打标/取消，目录 ★ 与进度条同步更新（localStorage `nce-mastered-lessons`）。
7. **单词逐个收藏**：单词表每行前置 ☆/★，点击加入/移出生词本（`/api/vocab/star|unstar`，与「单词」页共用数据）；乐观更新 + 失败回滚 + toast 反馈。
8. **🎧 听写本课**：一键跳到听写页并自动预选本课开始（`NCE.pendingDictation` 深链，feat-dictation.js 配合消费）。

## 数据对齐修正

新概念第一册奇数课为课文、偶数课为配套练习，题目可能挂在任一课号。新增 `lessonStatFor(book, lesson)`：第一册把第 n 课与第 n+1 课的练习统计合并归到课文课 n，目录角标与学习情况条共用。其余册不做合并。

## 改动文件

- `public/index.html`：目录页加进度条 + 搜索框；内容页 quiz-head 加"标为已掌握 / 听写本课"按钮组。
- `public/app.js`：`loadMasteredLessons/toggleLessonMastered`、`updateTocProgress`、`filterToc`、`lessonStatFor`、`accClass`、`updateMasterBtn`；`loadLessons` 并行拉课程 + 统计；`openLesson` 并行拉课文 + 生词本、渲染学习条与收藏星、绑定收藏事件。
- `public/js/feat-dictation.js`：课程下拉就绪后消费 `NCE.pendingDictation` 预选并自动开始（目前听写仅支持第一册，深链带 book 校验）。
- `public/styles.css`：`.toc-progress / .toc-search / .acc / .toc-item.mastered / .lesson-stats / .wstar / .lesson-actions` 等纸面风格样式。

## 备注

- 已学/已掌握存 localStorage，换浏览器不迁移；生词收藏存服务端 `data/vocab.json`，可随备份走。
- 听写模块目前只加载第一册课程（其内部写死 book=1），深链对非第一册课程会安全降级为仅打开听写页。
