# Change：新概念英音原声随项目分发 + 🔊 优先原声

**日期**：2026-07-04

## 需求

1. 接入本地「英音」MP3 + LRC，教材页可整课播放、点句跳转。  
2. 全系统 🔊 在能匹配 LRC 句子时优先播原声片段，而非 TTS。  
3. **不依赖某台电脑路径**：MP3 复制进仓库，Git LFS 托管，克隆即可用。

## 实现概要

### 数据与音频

- `scripts/import-nce-official.js`：从英音源目录解析 LRC，默认 **复制** MP3 到 `public/audio/nce/`。  
- `data/official/passages.json`：276 课（1 册 72 + 2 册 96 + 3 册 60 + 4 册 48）。  
- `public/audio/official-segments.json`：按 `book-lesson` 索引每句 `{n, audio, start, end}`，供前端片段匹配。  
- `.gitattributes`：`public/audio/nce/*.mp3 filter=lfs`（约 552MB）。

### 后端

- `routes/official.js`：`GET /api/official/meta`、`GET /api/official/:book/:lesson`。  
- `lib/nce-official-util.js`：LRC 解析、`normSpeakLine` 归一化。

### 前端

- 教材 **🎧 原声课文** 标签：整课播放器 + 句子高亮。  
- `speak()` 优先级：**原声片段 → Wikimedia 真人词 → TTS**。  
- 听写 `setSpeakContext({ book, lesson })` 供跨页匹配。  
- 「朗读全文」：有原声时播整课 MP3。

## 克隆注意

必须 `git lfs install` 后 clone，或 clone 后 `git lfs pull`，否则 MP3 为指针无法播放。
