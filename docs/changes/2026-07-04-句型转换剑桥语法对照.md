# 句型转换 × 剑桥语法在用对照

## 摘要

句型转换模块与《剑桥初级英语语法》（Essential Grammar in Use，EGIU）及《剑桥中级英语语法》（English Grammar in Use，EGIU2）建立单元对照，练习链与剑桥左页讲解顺序对齐，便于与 NCE 课文同步强化。

## 对照体系

| 文件 | 说明 |
|------|------|
| `data/reference/cambridge-grammar.json` | EGIU / EGIU2 单元 ↔ NCE 语法标签 ↔ 推荐转换顺序 |
| `lib/cambridge-grammar.js` | 推断单元、统计覆盖、生成提示语 |

| NCE 册 | 剑桥语法书 |
|--------|------------|
| 第 1 册 | EGIU（初级，约 40 个核心单元已映射） |
| 第 2–4 册 | EGIU2（中级，时态/被动/从句/非谓语等核心单元） |

## 功能

- **筛选模式**：句型转换页可选「语法点」或「剑桥单元」
- **单元 chips**：按剑桥章节分组（现在时、过去时…），显示每单元可用句数
- **练习标注**：答题卡显示 `📘 U6 一般现在时否定`；解析末尾附剑桥单元提示
- **EGIU 顺序**：部分练习按 Unit 6/11 惯例采用「中译英 → 否定 → 疑问 → 提问」链
