# Change：听写答案区显示中文译文

**日期**：2026-07-03

## 需求

用户建议：听写显示标准答案的地方，带上这句话的中文翻译，方便对照学习。

## 背景

听写句子取自课文的单词例句（`words[].eg`）和语法例句（`grammar[].examples`），原文多为"英文 + 中文释义"混排（如 *Excuse me, is this seat free? 打扰一下，这个座位空着吗？*）。原实现用 `enOnly()` 只保留英文用于朗读与批改，中文释义被丢弃，答案区看不到翻译。

## 改动内容（仅 `public/js/feat-dictation.js`）

- 新增 `cnOnly(s)`：从原句提取中文部分（汉字 + 中文标点）。
- `collectSentences()` 返回值由英文字符串数组改为 `{ en, cn }` 对象数组：`en` 用于朗读/批改（逻辑不变），`cn` 为译文（原句无中文则为空）。
- 新增 `answerHtml(s)`：答案区统一渲染"📖 标准答案：英文"+（若有）虚线分隔的"译：中文"。
- "显示答案"与"提交批改"两条路径都换用 `answerHtml`；朗读、比对、重听均改用 `s.en`。
- 样式加 `.dct-cn`（灰色小字 + 虚线上边框）。

`tokenize` / `compare` 纯函数及其 `module.exports` 未动，不影响既有测试。

## 验证

Puppeteer 端到端：从教材学习"听写本课"深链进入 Lesson 1 听写——

- 点"显示答案"：显示英文 + "译：打扰一下，这个座位空着吗？"；
- 提交批改：逐词 diff 下方同样显示英文答案 + 中文译文；
- 连续抽查前 5 句全部带有译文；无页面报错。
