# Change：教材学习补充——翻页、书签恢复、目录跟随、已学标记

**日期**：2026-07-03（书页式目录的第二轮补充，见 [2026-07-03-教材学习书页式目录.md](./2026-07-03-教材学习书页式目录.md)）

## 背景

书页形式落地后，按"把它当一本真的书来用"检查，补齐四个缺口：

1. **上一课 / 下一课翻页**：书页却不能连续翻页，每次都要回目录点。
2. **书签恢复**：首页已有"继续学习"入口，但直接点"教材学习"标签仍是空白占位，不记得上次读到哪。
3. **目录跟随**：翻页或从书签恢复后，目录高亮项可能滚出可视区。
4. **已学标记**：目录里分不清哪些课学过了。

## 改动内容（仅前端）

- `public/app.js`
  - 新增 `state.learnLessons`（当前册目录，供翻页），`navLesson(delta)` 上/下一课。
  - 课程详情底部新增书页页脚 `.page-nav`：上一课 / 页码（`Lesson X · 第 n / N 课`）/ 下一课；首末课对应按钮禁用。
  - 键盘 ←/→ 翻页（仅教材学习页可见且未聚焦输入框时生效）。
  - `loadLessons()` 渲染目录后，若 `nce-last-lesson` 书签属于该册则自动打开该课（与首页"继续学习"共用同一份书签数据）。
  - 新增 `nce-viewed-lessons`（localStorage）记录已学课程；`openLesson()` 打开即记为已学。
  - `openLesson()` 高亮目录当前课后 `scrollIntoView({block:'nearest'})`，目录跟随不脱节。

- `public/styles.css`
  - `.page-nav`：双线分隔的书页页脚，页码用衬线字体纸面配色。
  - `.toc-item.viewed .no::after`：已学课程课号后加绿色小 ✓（当前课上为暖棕色）。
  - 新增通用 `.btn:disabled` 置灰；内容页 ghost 按钮 hover 改纸面暖色。

## 验证

Puppeteer 端到端 10 步验证全部通过：

1–4. 原有用例（按册筛选、点目录打开、高亮、空册提示）；
5. 切回第 1 册后书签自动恢复到上次课程；
