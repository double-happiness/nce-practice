# 新功能：情景对话 + 跨册查词典

日期：2026-07-03

## 情景对话

「💬 情景对话」标签页：按生活场景分类的角色扮演练习。

- **9 个场景**：机场、咖啡馆、健身房、居家、办公室、公园、餐厅、购物、旅行（各 3 组对话，共 27 条）
- **练习方式**：系统展示对方台词 + 你的中文提示，输入英文作答，逐步判分（大小写/标点不敏感）
- **数据**：`data/dialogues/*.json` 分片，按场景一文件
- **进度**：`dialogues.json` 按档案持久化（练习记录 + 完成标记）

## 查词典

「📕 查词典」：合并第 1、2 册教材词表，支持：

- 跨册浏览与搜索（英文 / 中文 / 例句 / 音标，相关度排序）
- 🔊 发音、☆ 收藏进生词本
- 教材学习页单词行 📕 深链跳转并高亮
