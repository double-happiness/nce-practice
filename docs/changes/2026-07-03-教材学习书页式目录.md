# Change：教材学习目录改为书页风格

**日期**：2026-07-03（同日对"左侧目录"方案的视觉迭代）

## 需求

在左侧目录 + 按册筛选的基础上（见 [2026-07-03-教材学习左侧目录.md](./2026-07-03-教材学习左侧目录.md)），目录观感不佳，要求：

1. 做成类似"书页"的形式；
2. 点击左侧目录，右侧跳转到对应课时；
3. 当前选中的目录项要明显高亮，方便区分。

## 改动内容

### 前端（无后端改动）

- `public/index.html`
  - 教材学习区重构为 `.book`（翻开的书）：左页 `.toc-page` 为目录页（"目 录 / Contents" 页眉 + 目录列表），右页 `.content-page` 为内容页（占位提示 / 课程详情）。
  - 册数切换 chips 保留在书本上方。

- `public/app.js`
  - 目录条目改为书籍目录排版：英文标题 + 点线引导（`.dots`）+ 课号（页码位），下行为中文标题。

- `public/styles.css`
  - 移除原侧栏样式（`.learn-layout` / `.learn-sidebar` 等），新增书页样式：
    - 米色纸面（`#fbf7ec`）、右下角多层 box-shadow 模拟书页堆叠、中缝两侧 inset 阴影模拟折页；
    - 目录页/标题使用衬线字体（Georgia / Songti SC），页眉带双线分隔；
    - 目录项点线引导 + 页码式课号；hover 为淡金色背景；**当前课左侧红色书签条 + 米金色底 + 标题变深红加粗**，与备选项区分明显；
    - 目录列表在长目录时独立滚动（细滚动条），整页 sticky 跟随；
    - 内容页配色适配纸面：课文情景框、语法卡片、单词表分隔线、笔记框等改为纸面同系暖色；
    - 移动端（≤760px）书页上下堆叠，目录限高 42vh。

## 交互确认

- 点击左侧目录项 → 右页立即渲染对应课时并回到页首；目录中该课高亮（红书签 + 深红标题）。
- 切换册数 → 目录只显示该册课程；右页清空回占位提示；无数据的册显示"暂未收录"。
