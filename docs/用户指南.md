# 新概念英语练习系统 · 用户指南

> 应用内阅读：启动服务后打开 [http://localhost:3737/help.html](http://localhost:3737/help.html)  
> 维护者文档见项目根目录 [README.md](../README.md)

---

## 1. 快速开始

```bash
npm install    # 首次需要联网，仅安装 express
# 从 Git 克隆后需拉取原声 MP3（Git LFS，约 550MB）：
# git lfs install && git lfs pull
npm start      # 默认 http://localhost:3737
```

浏览器打开上述地址即可。学习记录保存在本机 `data/` 目录，**无需注册账号、无需联网**（服务启动后断网仍可使用，见下文 FAQ）。

## 2. 界面怎么组织

顶栏分为四组，对应学习阶段：

| 分组 | 包含功能 | 适合什么时候用 |
|------|----------|----------------|
| **学** | 📖 教材学习、📕 查词典 | 学新课、查词、看语法 |
| **练** | ✏️ 刷题、🎧 听写、🔀 句型转换、💬 情景对话、📝 阶段测验 | 巩固、输出训练 |
| **复习** | 🔁 间隔复习、🔤 单词（词表/背单词/词汇测） | 抗遗忘、补漏洞 |
| **我的** | 📊 薄弱分析、🎯 学习计划、💾 数据备份 | 看进度、定计划、备份 |

**🏠 今日学习**（首页）是调度中心：今日目标、到期错题、薄弱语法、词汇量、继续上次课程、快捷查词——建议每天从这里开始。

首次进入首页会显示简短欢迎引导；完整说明见 **📖 使用说明** 卡片或本页。

## 4. 推荐学习路径

### 4.1 零基础 · 第一册（约 45–60 分钟/天）

以 **当前 1 课** 为中心，完成「听→说→读→写→记」闭环：

| 顺序 | 功能 | 建议时长 | 目标 |
|------|------|----------|------|
| 1 | 📖 教材学习（含 🎧 原声） + ✏️ 练本课语法 | 15 min | 听懂原声、读懂课文、掌握语法点 |
| 2 | 🎧 听写（选本课） | 10 min | 精听 + 拼写 |
| 3 | 💬 情景对话 或 朗读精读 | 10 min | 开口、语调 |
| 4 | 🔀 句型转换 | 10 min | 中译英、句型改写 |
| 5 | 🔤 背单词 + 🔁 复习到期错题 | 10 min | 词汇 + 抗遗忘 |

- **周日**：不学新课，做 **📝 阶段测验**（本周 6 课区间），错题自动进复习。
- **第四册**：每学完 12 课可用预设「L1–12 / L13–24 …」20 题模考；全书复习可用「全书模考 · 40题」。
- **每月**：做一次 👂/📖 词汇量测试，在 **📈 词汇趋势** 看变化。
- 详细计划说明见应用内 **🎯 学习计划** 页。

### 4.2 有基础 · 第二册

- 教材学习按课推进；刷题/句型转换与当前课同步。
- 薄弱分析出现红色语法点时，用首页 **专练 10 题** 或刷题筛选该语法。
- 查词典展开详情，看固定搭配与课内句型。

### 4.3 只补某一技能

| 目标 | 优先用 |
|------|--------|
| 语法弱 | 刷题 + 句型转换 + 间隔复习 |
| 词汇弱 | 背单词 + 听/读词汇量 + 查词典 |
| 听力弱 | 听写 + 听力词汇量 |
| 口语弱 | 情景对话 + 听写跟读 |
| 考前复盘 | 阶段测验 + 薄弱分析 |

## 5. 常用操作提示

### 5.1 🔊 发音与原声课文

全站 **🔊** 自动按优先级选择音源：

1. **新概念英音原声**（与 LRC 课文句一致时）  
2. **真人单词发音**（单词/短短语）  
3. **浏览器 TTS**（以上都没有时）

**教材页 · 🎧 原声课文**：整课 MP3 播放，点击某句可跳转并高亮；1–4 册共 276 课原声随项目内置，**无需联网**。

顶栏 🔊 语速下拉只影响 TTS；原声按录音原速播放。  
详细说明见 [原声与发音.md](原声与发音.md)。

### 5.3 💬 情景对话（二级目录）

左侧 **场景目录** 为两层：

1. **大类**（8 个）：居家社交、餐饮购物、出行旅游、职场学习、住房物业、健康运动、生活服务、休闲文娱  
2. **具体场景**（73 类）：如「打电话」「IT工作」「机场」等  

点击大类展开/收起，再点场景，右侧列出该场景下的对话练习。顶部搜索框可搜 **大类名或场景名**。

### 5.4 其他技巧

- **查词**：课文单词点 📕；选中英文短语也会弹出查词；词典内按 `/` 快速聚焦搜索框。
- **深链**：查词典支持 `#dictionary?q=umbrella`；教材支持 `#learn?book=1&lesson=3`。
- **生词本**：词表/词典/教材 ☆ 收藏，在 **词表** 子页复习。
- **错词复习**：词汇测试后可「逐个查词典复习」；首页会提示继续进度。
- **多档案**：数据备份页可切换档案（家人共用一台电脑时）。

## 6. 数据、备份与离线

| 项目 | 说明 |
|------|------|
| 存哪里 | 本机 `data/profiles/<档案名>/`（进度、错题、生词、计划等） |
| 备份 | **💾 数据备份** 导出 JSON；建议定期导出 |

## 8. 常见问题 FAQ

**Q：打不开页面？**  
A：确认终端里 `npm start` 已运行，浏览器访问 `http://localhost:3737`。

**Q：刷新后进度还在吗？**  
A：在。进度写在本地 JSON；除非清空数据或删 `data/`。

**Q：没有声音？**  
A：检查系统音量与浏览器权限。课文句优先播 **原声 MP3**；单词播真人库或 TTS。TTS 需 Mac/Windows 已安装英文语音。若原声无法播放，可能是克隆时未执行 `git lfs pull`（见 [原声与发音.md](原声与发音.md)）。

**Q：🔊 为什么是机器声而不是原声？**  
A：内容与 LRC 课文句不完全一致时会降级为 TTS；单词单独点击通常播单词发音而非整句原声。

**Q：情景对话左侧显示「没有匹配的场景」？**  
A：硬刷新页面（`Cmd+Shift+R`）；若从 Git 克隆，确认已 `git lfs install && git lfs pull` 并重启 `npm start`。侧栏应显示 8 个大类，其下为具体场景。

**Q：两台设备怎么同步？**  
A：系统无云同步。用 **数据备份** 导出 JSON，拷到另一台机器导入。

**Q：词典里全局词为什么没有教材例句？**  
A：全局词库约 2600 词，仅教材词有课内例句与句型；展开会标注「全局词库 · 仅词库释义」。

**Q：如何清除「知道了」后的欢迎引导？**  
A：浏览器清除站点数据中的 `nce-guide-dismissed`；或删对应 localStorage 项后刷新。

**Q：我是开发者，如何加题/加课？**  
A：见 [README.md](../README.md)「如何扩充题库」与 `docs/changes/` 变更说明。
