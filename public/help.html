📖 使用说明

新概念英语 1–2 册配套练习系统：教材、刷题、听写、词典、句型转换与复习闭环。Markdown 版见仓库 docs/用户指南.md。

1. 快速开始

npm install    # 首次需联网
npm start      # 打开 http://localhost:3737

学习记录在本机 data/，无需账号。服务启动后断网仍可使用（不依赖云端）。

5. 🔊 发音与原声课文

全站小喇叭按优先级自动选音源：

  1. 新概念英音原声 — 与课文 LRC 句一致时（教材页 🎧 原声课文）
  2. 真人单词发音 — 单词 / 短短语
  3. 浏览器 TTS — 以上都没有时
📌 教材页打开 🎧 原声课文：整课 MP3 播放，点击句子跳转并高亮。 276 课原声已内置，离线可用。顶栏语速只影响 TTS。

8. 常见问题

打不开页面？
确认终端已运行 npm start，访问 http://localhost:3737。
没有发音？
检查音量。课文句优先原声 MP3；单词用真人库或 TTS。TTS 需已安装英文语音。
为什么是机器声不是原声？
内容与 LRC 课文句不一致时会用 TTS；单独点单词通常播词发音。
克隆后原声播不了？
需安装 Git LFS 并执行 git lfs pull，见 docs/原声与发音.md。
两台设备怎么同步？
无云同步。用数据备份导出 JSON，在另一台导入。
断网能用吗？
可以。本机服务在跑即可；不访问外网 API。
全局词为什么没有教材例句？
仅教材收录词有课内例句与句型；全局词展开会标注「仅词库释义」。
